# FNAF2 Foxy Jumpscare

A Stardew Valley mod that adds jumpscares to spice up your farming life.

![foxy1](https://github.com/user-attachments/assets/37bcc01c-6cdb-4121-b077-c0ff4fbf4117)

## Features

- **1 in 100,000 chance per game tick** (~60 ticks per second) to trigger a jumpscare at any time.
- **1 in 100 chance** to trigger a jumpscare when opening a chest.

## Requirements

- [SMAPI](https://smapi.io/) 4.0.0 or newer
- Stardew Valley (latest version recommended)

## Installation

1. Download the mod from [Releases](https://github.com/crysqrlkys/stardew-jumpscare/releases).
2. Extract the contents into your `Mods` folder.
